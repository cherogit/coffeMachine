# gulp#4 сборка
Заготовка под "нулевой" новый проект.

## Используемые в сборке инструменты:

Шаблонизатор HTML
```
Pug
```
Методология
```
BEM
```
Постпроцессор
```
Postcss
```

### Установка пакетов из зависимостей:
```
npm install
```

### Запуск сборки и сервера:
```
gulp dev
```

Завершить работу сборщика можно командой `Ctrl + C`.
